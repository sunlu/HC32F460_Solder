# UI更改
## 主界面
深色背景，信息分为顶部状态栏、中央仪表区、左/右/下三侧参数区，整体风格为工业仪器风，数字清晰，带进度条和温度曲线示意。

顶部状态栏（居中偏上）
最顶部中间：显示型号标识 C245（可能代表手柄/发热芯型号）

其下方：分为三列电压/电流/功率数值，字体较大且加粗：

左侧：Voltage 下方显示 24.1 V （当前输入电压）

中间：Current 下方显示 0.00 A  （当前手柄电流）

右侧：Power 下方显示 0.00 W （当前加热功率）

中央主要区域（占画面主体）
左侧纵向：一个温度进度条/柱状图，从下往上填充，刻度标注在左侧：

刻度值从下到上依次为：0、25、50、75、100（推测为功率百分比或温度百分比）

当前柱状填充高度约在 75~80 位置（浅蓝色或白色高亮条）

右侧纵向：另一个进度条，刻度从下到上为：0、120、240、360、480（推测为温度°C或功率W）

当前填充高度约在 320~350 区间

左侧参数区（中央偏左，横向排列）
显示三个通道/模式设定值，字体较大，带编号：

Ch1 右侧显示 300（无单位，推测为预设温度°C）

Ch2 右侧显示 320

Ch3 右侧显示 350

这些数值与右侧进度条的高度大致对应

右侧参数区（中央偏右）
一个醒目的设定温度显示框，带有红色或高亮边框：

内容为 SET 080°C

其中 080 为当前设定值，°C 为温度单位

底部区域
从左到右有多个小型数值或状态指示（可能为工作模式、时间、或加热状态），排列较密，字体偏小

底部最右侧或中间偏下位置，有一个圆形或方形指示灯，疑似表示加热输出状态（可能为绿色/红色）

色彩与字体风格
背景：深灰/黑色（工业屏风格）

主要数值：白色或亮蓝色，高亮

进度条：蓝色渐变 或 亮青色

设定温度（SET）：红色或橙色高亮

单位（V/A/W/°C）：灰色小字

屏幕比例与布局坐标参考（归一化 0–100）
顶部标题区：Y=0~15

电压/电流/功率三列：Y=15~35

中央进度条区域：Y=35~85（左柱 X=10~30，右柱 X=70~90）

Ch1~Ch3 参数：Y=40~60，X=35~55

SET 温度框：Y=40~60，X=60~80

底部状态区：Y=85~100
